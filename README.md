# Toolbox

Toolbox is a utility to quickly get the commands you need to install all your CLI tools at once.

## Contributing

Feel free to open a pull request to add any tools you think should be added. All tools are located in the `src/tool.json` file. Only tools that have widespread use and first-party installation scripts should be added. Basically, they should be tools that a lot of people currently use and the install script should come straight from the tool's website.

You may also submit pull requests to improve design, functionality, or fix any bugs.
